🎬 OTT Movies Data Analysis Project

## 📌 Project Overview

This project analyzes OTT movies data using Python, SQL, and Power BI to identify trends, ratings, votes, genres, and overall movie performance across streaming platforms.

## 🛠️ Tools & Technologies

 🐍 Python
 📊 Pandas
 📈 Matplotlib
 🎨 Seaborn
 🗄️ SQL
 📉 Power BI

## 🔄 Project Workflow

1. 🧹 Data Cleaning & EDA using Python
2. 🗃️ SQL Analysis Queries
3. 📊 Interactive Dashboard in Power BI
4. 💡 Insights Generation

## 🔍 Key Insights

 ⭐ Identified top-rated OTT movies
 🗳️ Analyzed relationship between votes and ratings
 🎭 Compared genre-wise movie performance
 📈 Visualized trends using Power BI dashboards

## 🐍 Python Analysis

✅ Performed:

 🔎 Missing value analysis
 🧾 Duplicate checks
 📊 Statistical summary
 🔗 Correlation analysis
 🌡️ Heatmap visualization
